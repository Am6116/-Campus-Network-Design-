# Campus Network Design

This project demonstrates a comprehensive university campus network design simulated using Cisco Packet Tracer.

## Project Overview
The network provides connectivity across multiple buildings (Computer Science, Business Administration, and Administration & Support). Key features include:
- Core routing and VLAN management.
- DHCP server configuration for automated IP assignment.
- Secure network infrastructure with Access Points and switch configuration.
- Data center setup including file storage and portal servers.

## Technologies Used
- Cisco Packet Tracer
- Network Infrastructure Planning
- IP Addressing & Subnetting
- Server Configuration (DHCP, NAS)

## How to View
To view this project, you need [Cisco Packet Tracer](https://www.netacad.com/courses/packet-tracer) installed. Download the `.pkt` file from this repository and open it directly.
